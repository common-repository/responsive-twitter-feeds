Add a nice twitter feeds on your wordpress website using Responsive Twitter Feed plugin easily.

== Description ==

Responsive Twitter Feed plugin enables you to embed twitter feeds easily on your wordpress website. You can change it's appearance by adding some css codes. The default style is a responsive one. In order to work this plugin, you will have to log into your twitter account and create an API key, and fill the settings form with respective details.

You can reffer this article of [twitter](https://dev.twitter.com/oauth/overview) to create your own API key. And you can see all the API keys created by you [here](https://apps.twitter.com/)

== Installation ==

1. Download the plugin zip package and extract it.
2. Put the folder named "responsive-twitter-feeds" under /wp-content/plugins/ directory.
3. Goto the plugins page in your WordPress admin panel and click "Activate"
4. Goto Settings -> Twitter Settings and enter your oAuth credentials (you would need to create a new application at https://dev.twitter.com/apps)
5. You can implements tweets into your posts and pages using shortcode [muaw_twitter_tweets]


For basic usage, you can also have a look at the [plugin homepage](http://makeusawebsite.lk/mplugins/responsive-twitter-feeds/).
